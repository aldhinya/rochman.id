# Personal landing page

A personal website built using Gatsby.

## Demo

Here is a working live demo : [https://rochman.id](https://rochman.id)

## Installation and usage

```
git clone https://github.com/aldhinya/rochman.id.git
cd rochman.id
npm install
gatsby develop
http://localhost:8000
```
